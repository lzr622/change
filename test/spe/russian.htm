Линия по дроблению камня

Технический процесс

Камни будут отправлены на вибропитатель, а затем перейдут в щековую дробилку для первичного дробления. Затем материалы будут отправлены на гидравлическую конусную дробилку вибропитателем для вторичного дробления. Вибрационный грохот будет отображать качественные материалы, а затем они будут отправлены на роторную дробилку вертикального вала для формовки. Другие материалы с большим размером не будут отправляться на конусную дробилку для переработки, пока эти размеры не будут соответствовать требованиям. Конечные продукты должны быть снова экранированы, а ленточный конвейер отправит их в законченную зону.

Линия по измельчению руд

Технический процесс

Во-первых, большие материалы должны обрабатываться щековой дробилкой. И тогда лифт отправит материалы, размер которых соответствует требованиям к бункеру для хранения. Электромагнитный виброподавитель транспортирует материалы в камеру измельчения для равномерного измельчения. И они отправятся в сепаратор для классификации с воздушным потоком вентилятора. Из-за импеллера классификатора некачественные материалы будут поступать в камеру измельчения для переработки, а порошок, размер которого подходит, войдет в циклон для сбора. Затем окончательный порошок будет выпускаться разгрузочным устройством.
